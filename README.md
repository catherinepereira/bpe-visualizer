## BPE Visualizer

This project is heavily inspired by the [Tiktokenizer](https://tiktokenizer.vercel.app) tokenization playground, which implements OpenAI's [tiktoken](https://github.com/openai/tiktoken) tokenization.

This project currently implements the modified BPE algorithm used in the GPT-2 tokenizer. Primarily, this is a visualization/educational tool, as I couldn't get the idea out of my head of how it would look to "step through" the BPE algorithm. No plans currently to support all of the tiktoken encodings.
